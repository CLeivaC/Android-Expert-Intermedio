## Descripción
La Horóscopo App es una aplicación móvil diseñada para brindar horóscopos diarios, semanales y mensuales de manera interactiva y entretenida. Esta aplicación combina la precisión astrológica con una experiencia de usuario atractiva, incluyendo animaciones y características personalizadas para cada signo del zodíaco.

## Características principales
- Horóscopos diarios, semanales y mensuales para cada signo del zodíaco.
- Animaciones interactivas que acompañan las predicciones astrológicas.
- Consejos astrológicos para mejorar la vida diaria según el signo del usuario.

## Tecnologías utilizadas
- Arquitectura MVVM y clean code para una estructura de código clara y mantenible.
- Fragments para la creación de interfaces de usuario modulares y reutilizables.
- Navigation Component para una navegación más sencilla entre los distintos fragmentos de la aplicación.
- Gradle KTS para la configuración del proyecto de manera más eficiente y con un lenguaje Kotlin más poderoso.
- Inyección de dependencias para una gestión eficiente de las dependencias y la modularidad del código.
- StateFlow y corrutinas para manejar el estado de la aplicación y operaciones asincrónicas de manera reactiva.
- RecyclerView para la presentación de listas de horóscopos y otros elementos en la interfaz de usuario.
- Retrofit, interceptors y mappers para realizar solicitudes HTTP a servidores externos y mapear los datos de respuesta.
- Intents para la comunicación entre diferentes componentes de la aplicación, como el inicio de actividades y servicios.
- Camera X para la integración de funcionalidades de cámara, por ejemplo, para escanear códigos QR o tomar fotos de usuarios.
- Animaciones para mejorar la experiencia del usuario, mediante el uso de bibliotecas como Lottie.
- UnitTest y UITest para asegurar la calidad del código y la funcionalidad de la aplicación mediante pruebas automatizadas.
